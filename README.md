# LeaderBoard

Leaderboard website displays scores submitted by different players and also allows submission of scores.Built with HTML,CSS, JavaScript, JS modules, webpack, API.

## Built With

- HTML
- CSS
- JS
- JS modules
- Webpack
- API

## Getting Started

To get a local copy up and running follow these simple example steps.

- In your terminal, move the location you want to clone the project.
- Enter git clone git@github.com:kemigabocatherine/LeaderBoard.git
- cd to the repo.
- Open the project and use it as desired
- **PS:** Use webpack to run and build project both locally and on the browser.
